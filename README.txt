# The Empire Affiliate Marketing Hub - Website Package

## What's Included

This package contains everything you need to deploy your website.

### Files

```
empire-website/
├── index.html                          ← Main landing page
├── about.html                          ← About Us page
├── terms.html                          ← Terms of Service page
├── README.txt                          ← This file
└── images/
    ├── logo-main.jpg                   ← Your crown logo
    ├── dashboard-welcome.png           ← Dashboard screenshot
    ├── dashboard-performance.png       ← Dashboard screenshot
    └── unstoppable.png                 ← Growth graphic
```

## How to Deploy

### Option 1: Upload to Web Hosting

1. Extract this ZIP file
2. Upload ALL contents to your web server root folder
3. Ensure the `images/` folder is in the same directory as the HTML files
4. Visit: https://www.empireaffiliatemarketinghub.com

### Option 2: Test Locally

1. Extract this ZIP file
2. Double-click `index.html` to open in your browser
3. Images should display immediately (no internet required)
